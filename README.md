# logg
read the CHAOS; shows the CLARITY

A lightweight Linux-only diagnostic tool that monitors kernel panics, detects system anomalies, and generates clean human-readable reports. It collects logs from multiple sources, analyzes failures, encrypts & stores them in secure ZIP archives, and can automatically push reports to a private GitHub repo for audit and backup.
Simple commands. Powerful insights. Developer-friendly crash forensics.

logG starts from Greatness !
